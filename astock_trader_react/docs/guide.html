📈 A股量化交易系统

智能选股 · 策略回测 · 风险控制

🏠 系统概览

本系统是一款面向A股市场的量化交易辅助工具，专注于尾盘策略——在交易日尾盘筛选强势股票，次日开盘冲高卖出，追求隔日高开溢价。

核心特点

  • 智能选股：基于多维度指标自动筛选候选股票
  • 策略回测：历史数据验证策略有效性
  • 实时监控：自选股价格变动提醒
  • 风险控制：内置止损止盈机制

⚙️ 运行逻辑

📊 数据采集 ↓ 🔍 尾盘选股 ↓ 📈 技术分析 ↓ 🎯 信号生成 ↓ 📋 策略回测 ↓ ✅ 决策执行

每日工作流程

  1. 14:30-15:00：系统扫描当日涨停股，筛选符合条件的标的
  2. 14:50-15:00：在尾盘确认信号后买入
  3. 次日 9:15-9:30：观察集合竞价，高开即卖出
  4. 次日 9:30-10:00：如未高开，择机止损或持仓

🖥️ 前端功能

1. 仪表盘

展示账户核心指标：

  • 总资产当前账户总市值
  • 累计收益历史总体收益率
  • 胜率盈利交易占比
  • 最大回撤历史最大亏损幅度
  • 夏普比率风险调整后收益

2. 选股

按策略条件筛选候选股票，支持：

  • 日期范围选择
  • 策略切换（尾盘策略等）
  • 股票详情查看（K线、指标）
  • 一键加入自选

3. 回测

验证策略历史表现：

  • 设置回测区间（起止日期）
  • 配置初始资金
  • 查看收益曲线、交易明细
  • 对比不同策略表现

4. 自选股

管理关注的股票：

  • 添加/删除自选股
  • 设置提醒价格
  • 查看持仓成本（手动记录）

5. 配置

调整策略参数：

  • 换手率阈值
  • 流通市值上限
  • 振幅控制
  • 止损/止盈比例

📖 名词解释

策略相关

  • 尾盘策略：在交易日最后30分钟买入，次日开盘卖出的超短线策略
  • 均线多头：短期均线上穿长期均线（如5日>10日>20日），表示上涨趋势
  • MACD金叉：DIF线上穿DEA线，看涨信号
  • 平台突破：股价横盘整理后放量突破前期高点
  • 量比：当日每分钟成交量/过去5日每分钟成交量，>1.5表示放量

选股条件

指标取值说明
昨日涨停必须有当日最强信号
量比>1.2尾盘放量，资金抢筹
换手率>3%流动性好，进出方便
流通市值<200亿盘子小，拉升容易
振幅<5%当日波动不能太大
股价4-30元太高门槛高，太低股性差

风险控制

  • 止损：亏损达-3%~-5%时无条件卖出
  • 低开止损：次日低开超过-2%立即清仓
  • 高开止盈：次日高开即卖出，不贪心

绩效指标

  • 总收益率：(期末净值-期初净值)/期初净值
  • 年化收益率：按一年252交易日折算的年收益
  • 夏普比率：(收益率-无风险利率)/波动率，越高越好
  • 最大回撤：从最高点到最低点的最大亏损
  • 胜率：盈利交易次数/总交易次数
  • 盈亏比：平均盈利/平均亏损
